SWADroid

Antonio Cañas Vargas, 2011-2015

Application for Android devices

SWADroid is an app for Android devices. You can download it for free from Google play.

Authors

SWADroid has been programmed by:

  • Juan Miguel Boyero Corral (leader and main developer)
  • Antonio Cañas Vargas (server side and icons)
  • Antonio Aguilera Malagón (module for control of attendance)
  • Helena Rodríguez Gijón (modules for file browser/download and registration in groups)
  • José Antonio Guerrero Avilés (module for course info)
  • Alejandro Alcalde Barros (interface and module for authentication)

Source code

SWADroid is free software. The source code is available in GitHub forge (https://github.com/Amab/SWADroid) under GNU GPLv3 and GNU LGPLv3 licenses.
